# Bangumi-Archive-Anime-Excel.xlsx

## 数据来源说明

本数据集基于Bangumi开源Archive文档的条目(Subject)数据,提取type=2的动画类型条目,并转换为Excel格式。原始字段定义如下:

### 条目(Subject)字段定义

| Key | 含义 | 备注 |
|-----|------|------|
| id | 条目ID | 唯一标识 |
| type | 作品类型 | 1表示漫画,2表示动画,3表示音乐,4表示游戏,6表示三次元(本数据集仅包含type=2动画) |
| name | 条目名 | 原始名称(日文/英文名) |
| name_cn | 条目简体中文名 | 简体中文名称 |
| infobox | 条目原始wiki字符串 | 包含动画的详细信息,如话数、片长、制作人员等 |
| platform | 条目平台 | 剧场版/TV/OVA/Web等播放平台 |
| summary | 条目简介 | 动画剧情概述 |
| nsfw | 是否为NSFW | Not Safe For Work,是否含有成人内容(布尔值) |
| date | 发行日期 | 动画发行日期 |
| favorite | 收藏状态 | 想看、看过、在看、搁置、抛弃 |
| series | 是否为系列作品 | 是否为系列作品(单行本等)(布尔值) |

### 2023-07-27起额外导出数据

| Key | 含义 | 备注 |
|-----|------|------|
| tags | 标签(部分) | 动画相关的标签列表 |
| score | 评分 | 综合评分(通常为0-10) |
| score_details | 评分细节 | 包含各个评分级别的分布详情 |
| rank | 类别内排名 | 动画在类别中的排名 |

### 2024-08-30起更新

- 正确处理条目名、中文名和原始wiki的HTML转义

### 2025-04-18起额外导出数据

| Key | 含义 | 备注 |
|-----|------|------|
| meta_tags | 公共标签 | 由维基人管理的标签 |

## 数据字段结构

本Excel数据集包含以下字段,全面记录每部动画的详细信息:

### 基础信息字段

| 字段名 | 含义 | 数据类型 | 来源版本 |
|--------|------|----------|----------|
| **id** | 条目ID | 数字/字符串 | 初始版本 |
| **type** | 作品类型 | 数字 | 初始版本(固定为2,表示动画) |
| **name** | 条目名 | 字符串 | 初始版本 |
| **name_cn** | 条目简体中文名 | 字符串 | 初始版本 |
| **platform** | 条目平台 | 字符串 | 初始版本 |
| **summary** | 条目简介 | 长文本 | 初始版本 |
| **nsfw** | 是否为NSFW | 布尔值/数字 | 初始版本 |
| **date** | 发行日期 | 日期(YYYY-MM-DD) | 初始版本 |
| **favorite** | 收藏状态 | 字符串 | 初始版本 |
| **series** | 是否为系列作品 | 布尔值/数字 | 初始版本 |

### 原始信息字段

| 字段名 | 含义 | 数据类型 | 来源版本 |
|--------|------|----------|----------|
| **infobox** | 条目原始wiki字符串 | 长文本/HTML | 初始版本(2024-08-30起正确处理HTML转义) |

### 评分与排名字段

| 字段名 | 含义 | 数据类型 | 来源版本 |
|--------|------|----------|----------|
| **tags** | 标签(部分) | 字符串/数组 | 2023-07-27 |
| **score** | 评分 | 数字(0-10) | 2023-07-27 |
| **score_details** | 评分细节 | 字符串/JSON | 2023-07-27 |
| **rank** | 类别内排名 | 数字 | 2023-07-27 |

### 元数据字段

| 字段名 | 含义 | 数据类型 | 来源版本 |
|--------|------|----------|----------|
| **meta_tags** | 公共标签 | 字符串/数组 | 2025-04-18 |

## infobox字段解析说明

infobox字段包含条目原始wiki字符串,其中可能包含以下信息(通过解析可提取):

- 话数
- 片长
- 制片国家/地区
- 语言
- 官方网站
- 放送开始时间
- 播放结束时间
- 动画制作公司
- 导演
- 音乐
- 人物设定
- 机械设定
- 原作
- 企画
- 制片人
- 动画制片人
- 脚本
- 分镜
- 演出
- 作画监督
- 美术监督
- 色彩设计
- 摄影监督
- 音响监督
- 主题歌作曲
- 主题歌作词
- 主题歌演出

注意:这些详细信息包含在infobox字段中,本数据集的Excel格式可能已将其解析为独立字段,具体取决于数据处理方式。
