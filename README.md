## 数据字段结构

本Excel数据集包含以下40个字段,全面记录每部动画的详细信息:

### 基础信息字段

| 字段名 | 说明 | 数据类型 |
|--------|------|----------|
| **id** | 动画条目的唯一标识ID | 数字/字符串 |
| **name_cn** | 动画中文名称 | 字符串 |
| **name** | 动画原始名称(日文/英文名) | 字符串 |
| **别名** | 动画的其他名称/别名 | 字符串 |
| **类型** | 动画类型(TV/OVA/电影/其他) | 字符串 |
| **话数** | 动画集数 | 数字 |
| **片长** | 每集时长(分钟) | 数字 |
| **制片国家** | 制作国家/地区 | 字符串 |
| **语言** | 播放语言 | 字符串 |
| **官方网站** | 动画官方网站链接 | URL字符串 |
| **platform** | 播放平台 | 字符串 |
| **URL** | Bangumi条目链接 | URL字符串 |

### 时间信息字段

| 字段名 | 说明 | 数据类型 |
|--------|------|----------|
| **放送开始** | 动画首播日期 | 日期(YYYY-MM-DD) |
| **播放结束** | 动画播放结束日期 | 日期(YYYY-MM-DD) |
| **date** | 数据收录日期 | 日期(YYYY-MM-DD) |

### 制作团队字段

| 字段名 | 说明 | 数据类型 |
|--------|------|----------|
| **动画制作** | 动画制作公司 | 字符串 |
| **导演** | 动画导演 | 字符串 |
| **音乐** | 音乐制作/作曲家 | 字符串 |
| **人物设定** | 角色设计 | 字符串 |
| **机械设定** | 机械设计 | 字符串 |
| **原作** | 原作作者/原作作品 | 字符串 |
| **企画** | 企划 | 字符串 |
| **制片人** | 制片人 | 字符串 |
| **动画制片人** | 动画制片人 | 字符串 |
| **脚本** | 编剧/剧本 | 字符串 |
| **分镜** | 分镜/分镜监督 | 字符串 |
| **演出** | 演出 | 字符串 |
| **作画监督** | 作画监督 | 字符串 |
| **美术监督** | 美术监督 | 字符串 |
| **色彩设计** | 色彩设计 | 字符串 |
| **摄影监督** | 摄影监督 | 字符串 |
| **音响监督** | 音响监督 | 字符串 |
| **主题歌作曲** | 主题曲作曲 | 字符串 |
| **主题歌作词** | 主题曲作词 | 字符串 |
| **主题歌演出** | 主题曲演唱者 | 字符串 |

### 评分与数据字段

| 字段名 | 说明 | 数据类型 |
|--------|------|----------|
| **favorite** | 收藏人数/喜爱人数 | 数字 |
| **rank** | 动画排名 | 数字 |
| **score** | 综合评分 | 数字(通常0-10) |
| **score_details** | 评分详情 | 字符串/JSON |
| **关联的动漫ID** | 相关联的其他动画ID | 数字/字符串 |

### 元数据字段

| 字段名 | 说明 | 数据类型 |
|--------|------|----------|
| **summary** | 动画简介/剧情概述 | 长文本 |
| **nsfw** | 是否包含成人内容 | 布尔值/数字 |
| **tags** | 动画标签列表 | 字符串/数组 |
| **meta_tags** | 元数据标签 | 字符串/数组 |
